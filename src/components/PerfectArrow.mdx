# Perfect Arrow

Perfect arrow mathematically fills a given `width` and `height` perfectly. It guarantees a pointed vertex and curved stems.

<Props of={PerfectArrow} />

## Usage

<Playground>
  <PerfectArrow
    width={25}
    height={50}
    strokeWidth={5}
    color={"#ccc"}
    direction={"right"}
    // Optional
    // - Any remaining props get passed through to first elem
